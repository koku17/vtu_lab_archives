# Java Lab Programs AIML

## Prerequisites
You need to have `openjdk` and `make` installed on your system and it must be added to `PATH` variable

## How to run
To compile classes

Change Directory
```
cd <dir>
```

Here `<dir>` is `demo` or `Lab`

```
make build
```

To comile and run classes
```
make run
```

To remove compiled classes
```
make clean
```
